#  Rock, Paper, Scissors

A Rock Paper Scissors game application built with HTML, CSS, and JavaScript.
Play against the computer, and be the first player to reach 3 points to win the match.

## Rules of the game

- Rock beats Scissors
- Paper beats Rock
- Scissors beats Paper

## ✨ Features

- Play against the computer
- Live score tracking
- Animated choice reveal
- First player to reach 3 point wins the match
- Automatic game reset after each match
- Interactive hover effect

## 🕹 How to Use

1. Click Rock, Paper or Scissors to make your choice
2. Wait for the computer to make it's choice
3. The round winner will be displayed and the score is updated
4. The first player to reach 3 points wins the match
5. Game will automatically reset after match ends
